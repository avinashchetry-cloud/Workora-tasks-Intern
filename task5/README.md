# 📊 Global Superstore Interactive Dashboard

An interactive business intelligence dashboard built using **Tableau** to analyze the Global Superstore dataset. The dashboard provides insights into sales performance, profitability, shipping trends, and geographical distribution through interactive visualizations.

---

## 📌 Project Overview

This project focuses on transforming raw sales data into meaningful business insights using Tableau. The dashboard allows users to explore performance across different countries, states, categories, and shipping modes with interactive filters.

---

## 🎯 Objectives

- Analyze global sales and profit performance.
- Identify geographical sales patterns.
- Understand shipping mode efficiency.
- Track monthly sales trends.
- Build an interactive dashboard for business decision-making.

---

## 🛠️ Tools & Technologies

## 📂 Dataset

**Dataset:** Global Superstore

The dataset contains information about:

- Profit
- Quantity

- Geographic Locations

---

## 🧹 Data Cleaning

The following preprocessing steps were performed:

- Removed duplicate records
- Handled missing values
- Corrected data types
- Standardized date formats
- Removed unnecessary columns (e.g., Postal Code)
- Verified geographic fields for mapping
